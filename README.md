## ⚡ The Cyber-Editorial Design System

The platform balances soft, organic print-editorial canvas backgrounds with futuristic electric highlights:

* **Oatmilk Base Canvas:** `#f7f5f0` (Warm, sophisticated paper texture)
* **Rich Ink Typography:** `#1c1c1a` (High-contrast editorial reading layout)
* **High-Intensity Cyber Lines:** * `--neon-cyan`: `#00f3ff` (Glow: `0 0 12px rgba(0, 243, 255, 0.85)`)
    * `--neon-pink`: `#ff007f` (Glow: `0 0 12px rgba(255, 0, 127, 0.85)`)

### Architecture Features
1. **Linear Interpolation Cursor Engine:** Custom smart-smoothing halo custom mouse environment (`requestAnimationFrame` logic) with reactive magnetic states when hovering over elements.
2. **Top Scanline Tracker:** Fluid edge-to-edge gradient top banner animating key focal boundaries.
3. **Scroll Reveal Animation Engine:** IntersectionObserver API with a `0.12` threshold and cubic-bezier transition matrix (`cubic-bezier(0.2, 0.8, 0.2, 1)`) ensuring perfectly fluid viewport entrances.
